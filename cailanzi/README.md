## 居中

1.盒子居中
``` html
   <!-- 必须是块 -->
    width:500px;
    margin-left:auto;
    margin-right:auto;
```

``` html
    width:500px;
    position:absolute;
    left:50%;
    margin-left:-250px;
```
2.盒子中内容居中

``` html
    
    text-align:center;
```
